🎬 GUIMINAMFLIX

Um projeto simples inspirado em plataformas de streaming como a Netflix, desenvolvido com HTML e CSS, com o objetivo de praticar e evoluir habilidades em front-end.

## 📷 Preview do Projeto

![Preview do Projeto](./preview.png)

🚀 Sobre o projeto

O Guiminamflix é uma página web que exibe trailers de filmes e séries diretamente do YouTube, com um layout moderno e responsivo.

📚 Este projeto foi desenvolvido durante o ensino médio, utilizando a plataforma de estudos Alura, com foco no aprendizado prático de desenvolvimento web.

Durante o desenvolvimento, foram trabalhados conceitos como:

Estruturação com HTML
Estilização com CSS
Uso de fontes externas (Google Fonts)
Organização de layout com Flexbox
Incorporação de vídeos com iframe
